## 1. 개요: 왜 역전파 없이 학습하려 하는가?

현대 대규모 언어 모델(LLM)은 **Transformer + 대규모 데이터 + 역전파(BP)** 기반 최적화가 표준이다. 그러나 역전파에는 근본적 한계가 있다:

- **생물학적 비타당성**: Weight transport 문제, 시간 역전 등
- **메모리 병목**: 활성값 저장이 모델 크기에 비례
- **순차적 의존성**: 레이어 간 locking으로 병렬화 제한
- **미분 가능성 요구**: 미분 불가능한 목적함수 최적화 불가

이 서베이에서는 "역전파 없이" 언어 모델·시퀀스 모델을 학습하려는 연구를 **학습 신호의 전파 방식**에 따라 분류하고 비교한다.

---

## 2. "역전파 없이"의 스펙트럼

"비역전파 학습"은 단일 개념이 아니라 넓은 스펙트럼을 가진다:

| 분류 | 설명 | 대표 방법 |
|------|------|----------|
| **블랙박스 최적화** | 파라미터 변형을 병렬 평가, fitness만으로 업데이트 | EGGROLL (ES) |
| **근사 피드백** | 고정/학습된 피드백으로 근사 그래디언트 전달 | DSF, Feedback Alignment |
| **에너지 기반 로컬 규칙** | 자유상/클램프상의 평형 차이로 국소 업데이트 | EqProp |
| **온라인 그래디언트 대체** | 시간 역전 없이 온라인 갱신 | KF-RTRL |
| **구조적 회피** | 내부 가중치 고정, readout만 학습 | Reservoir/ESN |
| **로컬 신용할당** | 희소 구조 + top-k + 메모리 트레이스 | RSM/bRSM |

## 3. 주요 사례별 분석

### 3.1 EGGROLL — 초대규모 진화전략 (2026)

**접근**: 역전파를 완전히 제거하고, 진화전략(ES)으로 **추론 인프라를 학습 인프라로 전환**한다.

**모델**: EGG (Evolved Generative GRU) — int8 가중치 기반 6-layer GRU

**핵심 기법**:
- 저랭크(low-rank) perturbation으로 탐색 차원 축소
- 재생성 가능한 RNG로 perturbation 텐서 저장 회피
- population을 $2^{20}$까지 확장

**성능 (minipile 문자 예측)**:

| 방법 | Test bits/byte |
|------|---------------|
| EGGROLL (int8 RNN) | **3.40** |
| Backprop (fp32 Transformer) | 3.58 |

**비용**: 대규모 population 실험에서 backprop 대비 **~180배 GPU-hours**

**포스트트레이닝 응용**: 14B RWKV 7 모델에 DeepScaleR 데이터로 레시피 적용

> **평가**: 미분 불필요 + 통신량 최소화라는 장점이 있지만, 총 compute 폭증이 핵심 한계. "추론을 학습으로" 전환하는 아이디어는 포스트트레이닝/미분 불가능 목적에서 더 유망하다.

---

### 3.2 DSF — 고정 대각 피드백으로 RNN 학습 (2025)

**접근**: BPTT의 시간 그래디언트 전파를 완전히 끊는(FT-BPTT) 대신, **고정된 대각 피드백 행렬**로 시간 신용할당을 근사한다.

**핵심 결과 (Wikitext-103, ~28.2M params)**:

| 방법 | Validation PPL |
|------|---------------|
| BPTT | **28.32** |
| DSF | 31.54 |
| FT-BPTT | 46.60 |

BPTT 대비 격차는 있지만, 완전 절단(FT-BPTT) 대비 **크게 개선**된다.

**깊이별 확장 (Wikitext-103)**:

| 레이어 | Params | BPTT | DSF |
|--------|--------|------|-----|
| 1L | ~20.3M | 35.59 | 37.42 |
| 3L | ~28.2M | 28.32 | 31.54 |
| 12L | ~63.7M | 23.73 | 28.99 |

- 환경: 단일 A100
- 코드: GitHub 공개

> **평가**: "BPTT를 100% 버리진 않되 비용을 줄이는 근사"로 가장 실용적인 접근. 깊이와 시퀀스 길이 확장에서 일관된 성능을 보여준다.

---

### 3.3 RSM/bRSM — 로컬·즉시 신용할당 (2019)

**접근**: 희소한 구조(그룹-셀)와 top-k 마스크, 메모리 트레이스를 결합한 예측 오토인코더로, **완전 로컬·즉시적** 학습만 사용한다.

**결과 (Penn Treebank, next-word prediction)**:

| 방법 | Test PPL |
|------|---------|
| KN5 (baseline) | 143 |
| RSM | 166 |
| RNN (BPTT) | 124 |
| RSM+KN5 | 124 |
| **bRSM** | **103** |

bRSM이 BPTT-RNN보다 **낮은 PPL**을 달성했다는 점이 흥미롭다.

> **평가**: "비역전파가 backprop을 이길 수 있다"는 신호이지만, PTB는 소규모 코퍼스이므로 대규모 확장 검증이 필요하다.

---

### 3.4 EqProp — 평형전파로 NLP 분류 (2023)

**접근**: 에너지 기반 관점에서 수렴 RNN에 modern Hopfield attention을 통합, 평형전파로 학습한다.

**결과**:

| 데이터셋 | 정확도 |
|---------|-------|
| IMDB | 88.9 ± 0.3% |
| SNLI | 81.4 ± 0.2% |

- 코드, 실행 커맨드, 메모리 사용량까지 상세 제공
- 한계: 고정 임베딩 사용, 대규모 vocab softmax 미지원

---

### 3.5 기타 방법

| 방법 | 과제 | 핵심 수치 | 한계 |
|------|------|----------|------|
| **KF-RTRL** (2018) | PTB char-level | 1.72 BPC (vs BPTT 1.56) | 계산복잡도 $O(n^3)$ |
| **Target-Prop/ADMM** (2017) | PTB word-level | BPTT 근접 (설정 의존) | 대규모 확장 미검증 |
| **DNI 합성 그래디언트** (2017) | PTB char-level | 1.334 BPC (T=8) | 합성 모델 학습 필요 |
| **ESN** (2007) | 문법 구조 학습 | SRN 대비 문법 민감도 | 현대 LM과 비교 불가 |

## 4. 규모 확장 가능성 분석

### 계산량 관점

| 규모 | 유망한 접근 | 현실적 제약 |
|------|-----------|-----------|
| $10^7$ (수천만) | DSF, RSM/bRSM | 이미 검증됨 |
| $10^8$~$10^9$ (수억~수십억) | EGGROLL (포스트트레이닝) | 프리트레이닝 대체는 어려움 |
| $10^{10}$~$10^{11}$ (수백억~천억) | 없음 | 모든 비역전파 방식이 한계 |

**핵심 메시지**: "비역전파로 스케일한다" ≠ "경제적으로 스케일한다"

- EGGROLL: 병렬성 극대화 가능하지만 총 compute ~180배
- DSF: BPTT의 실용적 근사로 비용 절감
- 로컬 규칙: 메모리 효율적이지만 성능 격차

---

## 5. 연구 공백과 유망 방향

### 현재 공백

1. **대규모 LM 프리트레이닝에서의 공정 비교** 부족
2. ES 계열의 **비용-품질 트레이드오프** 체계적 축적 필요
3. EqProp의 **학습 가능 임베딩 + 대규모 vocab** 확장 미해결
4. 합성 그래디언트/타깃전파의 **현대 프레임워크 재검증** 부재

### 유망 방향

1. **추론 엔진 → 학습 인프라**: vLLM 같은 추론 커널을 학습에 재활용 (EGGROLL 방향)
2. **구조적 근사로 시간 신용할당 복원**: DSF의 대각 피드백처럼 완전 절단보다 강한 구조
3. **희소·메모리 + 로컬 규칙**: RSM/bRSM의 확장
4. **하이브리드 전략**: Hebbian 모듈(빠른 메모리) + 제한적 그래디언트 학습

## 7. 결론

역전파 없이 언어 모델을 학습하는 연구는 **이론적으로 매력적이지만 실용적으로는 아직 큰 격차**가 있다. 다만:

- **포스트트레이닝/미분 불가능 목적**에서 ES의 활용 가능성 (EGGROLL)
- **BPTT 근사**를 통한 비용 절감 (DSF)
- **소규모에서 BPTT를 능가**하는 로컬 학습의 가능성 (bRSM)

이 세 축에서 의미 있는 진전이 이뤄지고 있으며, 특히 뉴로모픽 하드웨어의 발전과 맞물리면 새로운 국면이 열릴 수 있다.
